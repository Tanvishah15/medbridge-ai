# Contributing to MedBridge AI

Thank you for your interest in this project.

## Synthetic Data Only

All medical reports, patient records, and knowledge documents in this repository are **fabricated for hackathon demonstration**.

- Do **not** upload real patient information (PHI/PII)
- Do **not** use real names, emails, phone numbers, or medical record IDs
- Use synthetic IDs only (e.g. `PAT-001`, `RPT-001`)

## Pull Requests

1. Keep changes focused and well-scoped
2. Ensure no secrets in commits (`.env` must stay local)
3. Test scripts in `scripts/` should pass before submitting

## Questions

Open an issue on GitHub for bugs or feature suggestions.
